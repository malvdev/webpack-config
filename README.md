# Webpack Configuration

This is webpack v5 configuration, that I am using for static projects.

## Installation

```bash
npm install
```

## Usage

### Development

```bash
npm run start
```

### Production

```bash
npm run build
```

### Test

```bash
npm run test
```

### Lint

```bash
npm run lint
```
